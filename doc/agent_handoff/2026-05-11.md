# 2026-05-11 MIPS Logisim 프로젝트 handoff

## 목적

32-bit pipelined MIPS processor를 Logisim으로 구현하는 과목 프로젝트 방향 정리입니다. instruction set은 `doc/mips_rv32i_instruction_reference.md` 기준으로 확정된 상태입니다.

사용자는 Verilog HDL로 RISC-V pipelined CPU를 설계한 경험이 있지만, Logisim 사용 경험은 없는 상태입니다.

## 현재 결론

Verilog HDL로 먼저 설계한 뒤 합성 netlist를 Logisim에 그대로 가져오는 방식은 비추천입니다.

이유는 다음과 같습니다.

- 합성 netlist는 Logisim 회로 구조와 자연스럽게 대응되지 않습니다.
- FPGA primitive, LUT, gate-level cell 중심으로 변환될 가능성이 높아 사람이 읽기 어렵습니다.
- pipelined CPU 과제에서 중요한 PC, pipeline register, control signal, forwarding, stall, flush 동작을 눈으로 추적하기 어렵습니다.
- 교수자나 조교가 구조를 확인하는 과제라면 의도와 맞지 않아 감점 위험이 있습니다.

따라서 Verilog는 설계 참고용 또는 reference simulator로 쓰고, 최종 제출 회로는 Logisim에서 구조적으로 직접 구현하는 방향이 안전합니다.

## 추천 진행 순서

1. 확정 instruction set을 기준으로 decode/control을 시작합니다.
   - R-type ALU: `add`, `addu`, `sub`, `subu`, `and`, `or`, `xor`, `nor`, `slt`, `sltu`
   - Shift: `sll`, `srl`, `sra`, `sllv`, `srlv`, `srav`
   - I-type: `addi`, `addiu`, `andi`, `ori`, `xori`, `slti`, `sltiu`, `lui`
   - Load/Store: `lb`, `lbu`, `lh`, `lhu`, `lw`, `sb`, `sh`, `sw`
   - Branch: `beq`, `bne`
   - Jump/Link: `j`, `jal`, `jr`, `jalr`
